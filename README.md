# 📐 Calcul Différentiel pour l'IA — Gradient & Hessienne

Ce TP explore les fondements mathématiques de l'optimisation utilisés en Machine Learning : gradient, matrice Hessienne, points critiques, et leur classification (minimum/maximum/point selle).

## 🔧 Outils
- Python
- SymPy (calcul symbolique)

## 📊 Contenu
- Calcul symbolique du gradient ∇f
- Matrice Hessienne et son déterminant
- Identification et classification des points critiques

## 🎯 Lien avec l'IA
Ces concepts sont au cœur des algorithmes d'optimisation comme la descente de gradient et la méthode de Newton, utilisés pour entraîner les modèles de Machine Learning.
